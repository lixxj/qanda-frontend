Dark/Light Mode Toggle

Description
Implemented a toggle for Dark/Light mode in the Qanda app, enhancing user experience by allowing preference-based theme selection. This feature supports visual comfort and accessibility.

Implementation
Theming: Utilized CSS custom properties for theme switching.

Persistence: Used local storage to remember user preferences across sessions.

User Interface: A toggle mechanism enables  theme changes.

Improves accessibility for users with light sensitivity.
Offers personalized user experience by allowing theme choice.
Enhances visual comfort in various lighting conditions.
